# IPLACEX Restaurant - Sitio Web Adaptativo

Este repositorio contiene el proyecto final de desarrollo y migración web para **IPLACEX Restaurant**, un sitio web interactivo y responsivo diseñado como una plataforma educacional y comercial para la escuela de gastronomía.

##  Enlace del Proyecto Desplegado
Puedes ver la página web funcionando en tiempo real aquí:  
 **https://sebastiancornejofinal.netlify.app**

## Descripción del Proyecto
El objetivo principal de este sitio web es servir como puente entre los estudiantes recién graduados de la escuela de gastronomía de nuestra institución y el mundo laboral. IPLACEX Restaurant es un establecimiento real/simulado creado a principios de 2026 que hoy cuenta con más de 30 trabajadores formados en nuestras aulas, especializándose en rescatar las raíces de la gastronomía 100% chilena.

### Características Principales:
* **Diseño Responsivo (Mobile-First):** Optimizado para celulares, tablets y computadoras gracias a Bootstrap.
* **Menú Tradicional Interactivo:** Separación por pestañas dinámicas de Desayunos, Almuerzos y Cenas típicas chilenas con imágenes optimizadas.
* **Sección de Reseñas:** Un carrusel de testimonios nativo que permite ver las calificaciones de clientes reales.
* **Identidad Corporativa Integrada:** Enlaces funcionales directos a las redes sociales oficiales de la institución que abren en pestañas independientes.

##  Decisiones de Diseño y Tecnologías Utilizadas
* **HTML5 & CSS3:** Para una estructura limpia y semántica.
* **Bootstrap 4:** Utilizado para la grilla adaptativa y el sistema de navegación dinámico.
* **FontAwesome:** Integrado mediante CDN para resolver problemas de fuentes locales y garantizar la visualización de íconos de estrellas, flechas y servicios en cualquier dispositivo.
* **Optimización de Rendimiento:** Se reemplazó el antiguo sistema de Slider Revolution (que generaba retrasos y pantallas en blanco por falta de dependencias) por una sección Hero fija basada en CSS puro, mejorando la velocidad de carga y la experiencia de usuario (UX).
